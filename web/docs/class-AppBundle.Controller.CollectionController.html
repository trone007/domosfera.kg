Class CollectionController

Контроллер отвечает за формирования коллекций и групп к коллекциям. Работает с сущностями ComplectData, Collection, CollectionGroup. Один артикул - может быть в нескольких коллекциях. Сохранение происходит в методе saveCollectionAction. Наименование коллекции отлично от наименования коллекций в 1С. Контроллер по сути наследник NotebookController.
